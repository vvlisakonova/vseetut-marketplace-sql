# vseetut-marketplace-sql
Разработка витрины данных и решение ad hoc задач на основе данных маркетплейса «ВсёТут». Включает формирование агрегированной витрины поведения пользователей и четыре аналитических SQL-запроса: сегментация пользователей, ранжирование по среднему чеку, статистика по регионам и анализ активности по месяцам.
